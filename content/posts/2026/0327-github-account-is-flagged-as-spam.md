title: '我的 GitHub 账号被封了'
---

2月22号的时候，我突然收到了一封邮件，关于我的一个开源项目的，问我是不是离开 Github 了，发邮件的是之前一起维护整个项目的国际友人，我当时有点莫名奇妙，因为在过年期间，我以为是因为我长期没有对该项目进行维护了，所以才会被误认为是离开了 Github 的，于是我就没有太在意

过了两天当我返回公司打开 Vscode 写代码的时候，突然注意到 Copilot 要求我重新登录，这时候才发现我的账号被标记了，只能登录以及一些基础的维护权限，但其他功能都无法使用———— Copilot, Sponsors, 修改账户信息等。同时，我的账户以及仓库对其他人来说是不可见的。我百思不得其解，只能提交工单

提交工单需要外国手机号，我26号提交了工单，当天晚上给我回复了，说是因为仇恨言论，所以被标记了，原因是我在一个知名的反华仓库中提的一个 issue，让我回复并作出保证，我回复后就没有下文了

现在我对我的博客和一些仓库进行了迁移，不知道还有没有机会恢复我的账户
